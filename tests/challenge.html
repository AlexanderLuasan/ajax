CSS Challenge

The Fantastic Wonders of CSS

Web technology has come so far in the past few years! There are now a plethora of ways to build webpages and it's often easy to forget how powerful plain CSS is!

This challenge serves as a demonstration of what can be accomplished through CSS-based design.

Bringing Life to Monotony

The thought of navigating the web without CSS is a scary one! In short, we’d be using a very basic looking web. The online world as we know it today couldn’t exist without CSS. An analogy commonly used is of building a house. While you could build a home only using beige bricks (HTML) it would have no features whatsoever! So, you could forget about all the colour on the walls, decoration, interesting architecture and visual effects :(

Lucky for us, a universally supported styling standard has been set to spare us the pain, thanks to the efforts of developers from the W3C and the major browser creators.

As a fun little experiment, try browsing the web with CSS styling turned off!

If you're up for the task, here's how you could disable CSS in various browsers:

  • Chrome: There isn't a built-in setting in Chrome to disable CSS, but it's possible using an extension like disable-HTML.
  • Firefox: View > Page Style > No Style
  • Safari: Safari > Preferences.. > Advanced > Show Develop menu in menu bar. Then, go to the Develop dropdown and select the “Disable Styles” option.
  • Microsoft Edge: Like Chrome, no built-in setting exists to do this. Luckily, the HTML Content Blocker extension is here to help!
  • Opera: We also need an extension for this one, and Web Developer is a good one!
  • Internet Explorer 11: View > Style > No style

Participation

Strong visual design is the focus of this challenge! You are going to be modifying this page using CSS ONLY. Get ready to put your CSS skills to the test!

You have exactly one hour to create your CSS design and submit it to qualify for the challenge.

At the end of this hour, your submissions will be judged by a panel of organizers and a single winner will be determined as the reigning champion of CSS-based design!

Feel free to check out the resource list below for references and guides on working with CSS.

Guidelines

You may modify the style sheet in any way you wish, but the HTML template must remain untouched.

Due to the huge number of user agents on the web these days, pixel-perfect layouts on all platforms may not be possible in such a short timeframe. That's okay, your design should work in at least the latest Chrome or Firefox desktop browsers.

Please submit a single CSS file titled "challenge.css" through the #css-challenge channel on discord before 8:00PM EST.

Don't forget to add your name and contact information (email and/or discord username) as a comment at the top of your file!

Tips & Tricks

This task may seem intimidating at first because you won't have as much control as you typically would as a web designer. So, here are some tips you may use to get the most options when styling a fixed html template:

  • use :first-child, :last-child and :nth-child to get at non-classed elements
  • use ::before and ::after to create pseudo-elements for extra styling
